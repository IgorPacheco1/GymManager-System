# GymManager-System

Sistema de gerenciamento para academias desenvolvido como trabalho prático para a disciplina de Banco de Dados (Semestre 1/2026) na PUC Minas.

## Descrição do Projeto
O GymManager-System é uma solução voltada para o controle operacional de academias. O sistema permite o gerenciamento de usuários, clientes, planos, matrículas e a prescrição personalizada de treinos, garantindo a organização dos dados e a agilidade nos processos de atendimento.

## Funcionalidades Principais
- **Gerenciamento de Acesso**: Autenticação de usuários para garantir segurança no acesso ao sistema.
- **Gestão de Clientes**: Cadastro completo de alunos e histórico de matrículas.
- **Controle de Treinos**: Registro de exercícios específicos por aluno, permitindo o acompanhamento de carga e séries.
- **Relatórios de Desempenho**: Consultas SQL avançadas para análise de dados e agregação.

## Tecnologias Utilizadas
- **Linguagem**: C# (.NET)
- **Banco de Dados**: MySQL (via MySQL Workbench)
- **Modelagem**: Diagrama Entidade-Relacionamento (DER) e Modelo Relacional usando PlatUML

## Estrutura do Repositório
- `/docs`: Contém a modelagem (DER, Modelo Lógico e Físico) em formato PDF.
- `/sql`: Scripts DDL (criação de tabelas) e DML (consultas SQL exigidas).
- `/src`: Código fonte da aplicação desenvolvida em C#.
